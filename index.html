Todo CLI Tool

The Todo CLI Tool helps manage tasks and reminders efficiently with features like Google Calendar integration and recurring task support. Stay organized on Linux, Windows, and Mac.

  • Manage tasks with reminders
  • Recurring task support
  • Cross-platform support
  • Integration with Google Calendar

Add a New Task

todo_task add --title "Team Meeting" --details "Discuss project updates" --start_time "2024-12-31T15:00:00Z" --end_time "2024-12-31T16:00:00Z" --recurring --frequency_minutes 1440

List All Tasks

todo_task list

Help

todo_task --help

    Installation

    # Example for Linux
tar -xvf todo-cli-tools-linux.tar.gz
sudo mv todo_task /usr/local/bin/
